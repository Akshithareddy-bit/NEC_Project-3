# 🏥 Healthcare AI System

A Smart Hospital Management System built using **Python, Streamlit, and SQLite**.  
This project helps manage patients, doctors, appointments, and electronic health records (EHR) in a simple and efficient way.

## 📌 Features

### 👨‍⚕️ Doctor Management
- Add doctors
- View doctor list
- Store specialization and experience

### 🧑 Patient Management
- Add patient details
- View patient records
- Link patients with doctors

### 📅 Appointment System
- Book appointments
- Track appointment status (Pending / Confirmed)

### 📄 Electronic Health Records (EHR)
- Store diagnosis
- Prescription details
- Doctor notes
- Visit history

### 🔐 Login System
- Role-based login (Admin / Doctor / Patient)
- Secure session handling using Streamlit session state
